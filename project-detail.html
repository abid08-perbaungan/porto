E-Commerce Mobile App

Mobile App
2024
3 bulan

Aplikasi e-commerce mobile yang dirancang untuk memberikan pengalaman berbelanja online yang seamless dan menyenangkan. Proyek ini mengimplementasikan berbagai fitur modern seperti pencarian produk, filter kategori, keranjang belanja, sistem pembayaran terintegrasi, dan tracking pesanan real-time.


Aplikasi ini dibangun menggunakan Flutter untuk memastikan performa optimal di kedua platform iOS dan Android. Backend menggunakan Firebase untuk authentication, real-time database, dan cloud storage. UI/UX dirancang dengan pendekatan user-centered design untuk memastikan kemudahan penggunaan.
